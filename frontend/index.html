Wie verwende ich diese Seite?

Auf dieser Seite können verschieden Betriebspunkte auf einer virtuelle Wärmepumpe simuliert werden.

Wie kann ich daten eingeben?

Parameter die umrahmt werden, wenn man mit der Maus drüber fährt, können verändert werden.
Wenn man auf einen Parameter klickt, öffnet sich ein Eingabefeld und man kann dort der Simulation einen Wert für den Parameter vorgeben.
Wenn man nun auf den "Simulieren" button drückt wird mit dem eingegebem Wert simuliert.
Das Eingabefeld kann durch einen Doppelklick wieder geschlossen werden.
